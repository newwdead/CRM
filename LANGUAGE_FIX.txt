   ```javascript
   <h1>⚙️ {t('adminPanelTitle')}</h1>
   <p>{t('adminPanelSubtitle')}</p>
   ```

3. ✅ Исправлена функция перевода в App.js:
   
   Было:
   ```javascript
   <AdminPanel t={{}} lang={lang} />  // пустой объект!
   ```
   
   Стало:
   ```javascript
   // Создана функция перевода
   const t = (key) => {
     return translations[lang]?.[key] || key;
   };
   
   <AdminPanel t={t} lang={lang} />  // правильная функция!
   ```

4. ✅ Добавлена перезагрузка при смене языка:
   
   ```javascript
   const toggleLanguage = () => {
     const newLang = lang === 'en' ? 'ru' : 'en';
     setLang(newLang);
     localStorage.setItem('lang', newLang);
     localStorage.setItem('language', newLang);
     window.location.reload();  // ← Перезагрузка страницы
   };
   ```

5. ✅ Импортирован translations в App.js:
   ```javascript
   import { translations } from './translations';
   ```
